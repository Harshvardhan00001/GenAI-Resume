# GenAI Resume Analyzer & Interview Prep

A powerful AI-driven tool that evaluates a candidate's resume against a specific Job Description (JD) and Software Developer (SD) role requirements. Powered by the Gemini API, it calculates a match score and automatically generates relevant technical and behavioral interview questions.

## 🚀 Features

* **Resume Scoring:** Compares user resumes against JDs to provide an accurate compatibility percentage.
* **Smart Parsing:** Extracts key skills, gaps, and experience using Google Gemini.
* **Technical Question Generator:** Generates tailored technical questions based on the gaps and skills found.
* **Behavioral Question Generator:** Mentorship/situational questions mapped to the role's seniority and culture fit.

## 🛠️ Tech Stack

* **Backend:** Node.js, Express.js
* **AI Integration:** Google Gemini API (`@google/generative-ai`)
* **Frontend:** React.js, Tailwind CSS (Optional / *Update based on your stack*)
